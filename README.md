# WiFi QR Generator

Generate a QR code for your WiFi so guests can scan and connect (iOS & Android, no app).

- Enter network name, password, security type
- Download PNG or print (full page + credentials)

## Run locally

Open `index.html` in a browser, or:

```bash
python -m http.server 8080
```

## Deploy to GitHub Pages

1. Create a repo on GitHub and push this folder.
2. **Settings → Pages** → Source: **Deploy from a branch**.
3. Branch: **main**, folder: **/ (root)**.
4. Site: `https://YOUR_USERNAME.github.io/REPO_NAME/`
